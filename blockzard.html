Blockzard

GitHub Repo

Blockzard is a captivating RPG where players, as Blockzard, take on quests to save the village. The game features dynamic NPC interactions, completing quests, strategic monster battles, and character progression through leveling up. Exploring an expansive world, players uncover hidden secrets and manage the village's well-being.
